## Technologies Used
FRONT-END
- javascript
- expo
- react-native
- native-base
- react-navigation
- redux

BACK-END
- javascript
- express
- nodejs
- postgreSQL
- jsonwebtoken

## Features
- Login / Sign up
- Home Screen with all comics
- Comic Detail
- Cart
- Profile
- Shipping Screen
- Order Screen
- Payment

## Dashboard Features
- Login
- E-commerce Summary
- Profile
- Orders, employees and customers pages.
- Apps (calendar, kanban, editor, color-picker)
- Analytics Charts (revenue, audience, sales, costs)
- Theme Settings (dark mode, colors)

<br>
Run 'expo start' inside client's folders.
